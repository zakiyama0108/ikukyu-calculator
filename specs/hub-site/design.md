# 設計: サイトのハブ化

## フォルダ構成（変更後）

```
app/
  layout.tsx              # 共通レイアウト（全ページ共通の HTML 骨格）
  page.tsx                # ハブトップページ（新規作成）
  ikukyu/
    layout.tsx            # /ikukyu 専用メタ情報
    page.tsx              # 育休給付金シミュレーター（app/page.tsx から移動）
  legal/
    page.tsx              # 変更なし
  components/             # 変更なし
  lib/                    # 変更なし
```

将来 `/kakeibo` 等を追加するときは `app/kakeibo/page.tsx` を追加するだけで対応できる。

## ハブトップページの構成

```
┌─────────────────────────────┐
│  ikukyu                      │  ← サイト名
│  育休・出産のお金をサポート     │  ← キャッチコピー
├─────────────────────────────┤
│  🧮 育休給付金シミュレーター   │  ← /ikukyu へリンク
│  産後からもらえる給付金が全部   │
├─────────────────────────────┤
│  （将来のツールカードはここ）   │
└─────────────────────────────┘
```

## メタ情報

| ページ | title | description |
|-------|-------|-------------|
| `/` | ikukyu \| 育休・出産のお金ツール | 育休・出産に関するお金のことをサポートするツールを提供しています。 |
| `/ikukyu` | 育休給付金シミュレーター \| ikukyu | 産後から育休まで、もらえる給付金が全部わかる無料シミュレーターです。 |

## 変更が必要なリンク

| ファイル | 変更内容 |
|---------|---------|
| `app/components/Footer.tsx` | 著作権表記を `ikukyu` に変更 |
| `app/legal/page.tsx` | `← シミュレーターへ` のリンクを `/` → `/ikukyu` に変更 |
| `app/layout.tsx` | グローバル metadata をハブ用に更新 |

## 決定事項

- サイト名: 「ikukyu」
- リダイレクト: 未インデックスのためリダイレクトなし
- Footer: 全ページ共通（app/layout.tsx 経由）のまま維持
