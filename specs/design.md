
### モード別の差異

| 項目 | ママ | パパ |
|------|------|------|
| 出産予定日ラベル | 出産予定日 | 配偶者の出産予定日 |
| 出産手当金カード | あり | なし（対象外バナーを表示） |
| 育児休業給付金（50%） | あり（181日以降） | 原則なし（産後パパ育休含む合計が181日以上になる場合は表示） |
| 出生時育児休業給付金 | なし | あり（産後パパ育休） |
| 出生後休業支援給付金（+13%） | あり（条件付き） | あり（条件付き） |
| 期間内訳バーの区分 | 産休 / 育休67% / 育休50% | 産後パパ育休 / 育休67% |

## データ型定義

```ts
// lib/types.ts

type Mode = 'mama' | 'papa'

type CalculatorInput = {
  mode: Mode
  monthlySalary: number    // 月給・円（手取り前の総支給額）
  dueDate: string          // 出産予定日（YYYY-MM-DD）
  leaveEndDate: string     // 育休終了予定日（YYYY-MM-DD）
}

type BenefitType =
  | 'maternity'            // 出産手当金（ママのみ・健康保険）
  | 'childcare67'          // 育児休業給付金 最初の180日（67%）
  | 'childcare50'          // 育児休業給付金 181日以降（50%、ママのみ）
  | 'paternity'            // 出生時育児休業給付金（パパのみ・産後パパ育休）

type BenefitItem = {
  type: BenefitType
  officialName: string     // 制度正式名（例：育児休業（育休）最初の180日）
  source: '健康保険' | '雇用保険'
  startDate: string
  endDate: string
  days: number
  rateLabel: string        // 表示用（例：「休業前賃金の67%」）
  amount: number
  bonusAmount?: number     // 出生後休業支援給付金の上乗せ額（+13%）
  dailyLimitReached: boolean
}

type PaymentSchedule = {
  startDate: string
  endDate: string
  days: number
  amount: number
  estimatedPaymentMonth: string  // 例：「2027年4月中旬ごろ」
  benefitType: BenefitType
  isFinal?: boolean        // 「最終振込」バッジ用
}

type BreakdownBarSegment = {
  label: string            // 例：「産休・98日」
  days: number
  colorClass: string       // Tailwindクラス（例：「bg-red-400」）
}

type CalculatorResult = {
  totalAmount: number
  summaryLabel: string     // 例：「出産手当金 + 育児休業給付金の合計」
  breakdownBar: BreakdownBarSegment[]  // 期間内訳バー（給付区分の日数割合）
  benefits: BenefitItem[]
  paymentSchedules: PaymentSchedule[]
}
```

## 計算ロジック方針

### ママ
| 給付金 | 財源 | 対象期間 | 給付率 |
|--------|------|---------|--------|
| 出産手当金 | 健康保険 | 産前42日 + 産後56日（計98日） | 標準報酬日額 × 2/3 |
| 育児休業給付金（前期）+ 出生後休業支援給付金 | 雇用保険 | 産後57日目〜84日目（最初の28日間） | 休業前賃金 × 67% + 13% = 80%（※1） |
| 育児休業給付金（前期） | 雇用保険 | 産後85日目〜育休180日目 | 休業前賃金 × 67% |
| 育児休業給付金（後期） | 雇用保険 | 育休181日目〜育休終了日 | 休業前賃金 × 50% |

※1 出生後休業支援給付金の+13%は、配偶者（パパ）が14日以上育休を取得していることが原則条件。ひとり親・配偶者が専業主婦（夫）・自営業の場合は単独取得でも対象。

### パパ
| 給付金 | 財源 | 対象期間 | 給付率 |
|--------|------|---------|--------|
| 出生時育児休業給付金 + 出生後休業支援給付金 | 雇用保険 | 育休1日目〜28日目（最大） | 67% + 13% = 80%（※1） |
| 育児休業給付金（前期） | 雇用保険 | 育休29日目〜育休180日目 | 休業前賃金 × 67% |
| 育児休業給付金（後期） | 雇用保険 | 育休181日目〜育休終了日 | 休業前賃金 × 50% |

※1 出生後休業支援給付金の+13%は、配偶者（ママ）が14日以上育休を取得していることが原則条件。例外あり（ママ側と同様）。
※2 180日のカウントは産後パパ育休（出生時育児休業）の日数も含めて通算する。パパが産後パパ育休28日 + 通常育休152日以上を取得すると50%期間が発生する。

### 上限・下限

上限額は毎年8月1日に改定される。以下は **2025年8月1日〜2026年7月31日** 適用の数値。

#### 雇用保険（育児休業給付金・出生時育児休業給付金・出生後休業支援給付金）

| 給付金 | 上限額 | 備考 |
|--------|--------|------|
| 育児休業給付金（67%） | **323,811円/月** | 賃金日額上限16,110円 × 30日 × 67% |
| 育児休業給付金（50%） | **241,650円/月** | 賃金日額上限16,110円 × 30日 × 50% |
| 出生時育児休業給付金（67%） | **302,223円**（最大28日分） | 賃金日額上限16,110円 × 28日 × 67% |
| 出生後休業支援給付金（+13%） | **58,641円**（最大28日分） | 賃金日額上限16,110円 × 28日 × 13% |

- 下限：雇用保険の最低賃金ベースで算出されるが実質的に極少額のため、UI上の下限チェックは省略してよい
- 上限到達時はカードの「日額上限」欄を「未到達」→「上限適用」に切り替えて表示

#### 健康保険（出産手当金）

| | 日額 | 根拠 |
|---|------|------|
| 上限 | **30,887円/日** | 標準報酬月額 上限（50等級・139万円） ÷ 30 × 2/3 |
| 下限 | **1,289円/日** | 標準報酬月額 下限（1等級・5.8万円） ÷ 30 × 2/3 |

- 実際の支給額は加入健保ごとに標準報酬月額の等級が決まるため、上記の上限・下限は計算上の理論値

### 振込スケジュールのロジック

```
振込予定月 = 対象期間の終了月 + 2ヶ月
振込時期   = 「中旬ごろ」（全給付金で統一）
```

- 育休給付金の申請単位は2ヶ月まとめが原則のため、振込スケジュールの区切りも2ヶ月単位にする
- 出生時育児休業給付金・出産手当金も同じ式で計算してよい（実績と一致）
- UIには「会社の手続きや状況により前後することがあります」の注意書きを表示する
